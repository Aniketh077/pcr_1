HOW TO START THE PROJECT
-------------------------

1. Start MongoDB:
   mongod

2. Start Backend Server:
   cd project/server
   npm install (if not done)
   node server.js

3. Start Frontend Development Server:
   cd project/ecotrade
   npm install (if not done)
   npm run dev

4. Access the application:
   - Frontend: http://localhost:5173
   - Backend API: http://localhost:5000

ADMIN ACCESS
------------

To access the admin panel:
1. Navigate to: http://localhost:5173/admin
2. Login with admin credentials (configured in your database)
3. Manage: Industries, Materials, and Buyer Requests

FEATURES AVAILABLE
------------------

Public Access (No Login Required):
- Browse industries
- Browse materials with dynamic filtering
- View material details
- Submit buyer requests (quote requests)
- Provide email OR phone number (at least one required)

Admin Access (Login Required):
- Dashboard with analytics
- Manage industries (CRUD)
- Manage materials (CRUD, stock management)
- Process buyer requests (status updates, notes, stock deduction)
- Export data to CSV

NO LONGER AVAILABLE (Removed):
- Email notifications
- Payment processing
- Shopping cart and checkout
- User registration for buyers
- Product orders
- Service requests (repair, recycle, sell)
- Business inquiries (separate from buyer requests)
- Newsletter subscription
- Contact form
- Wishlist functionality

The project is now a clean, focused B2B Eco Marketplace for PCR materials.
